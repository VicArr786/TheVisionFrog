# Apfel Grotezk family

**Collletttivo**  
*SIL Open Font License, 1.1*

Apfel Grotezk is a round and airy sans serif family, inspired by neo-grotesque and geometric typefaces.

#### Apfel Grotezk family contains:

* Regular
* Brukt
* Fett
* Satt

To contribute to the project contact [Collletttivo](https://www.collletttivo.it).
